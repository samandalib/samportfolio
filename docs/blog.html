4th Session

After exploring and learning basics of REST APIs, I tried to use it in a simple way. I made a small React component that convert currencies to each other. For doing the calculation I use the API feature of https://www.currencyconverterapi.com/. To test the API, at the following form, you can enter an amount of money (for example 10), then the base currency in capital letters (like UDS), and the last the desired currency in capital letters (like IRR). By clicking the convert button, the form will ask the api with the values you provided and the api will provide the calculated amount for you.
